## README

This is a sample Rails-3.2.7 application. It serves as an example API backend required to run [spree-on-react](https://github.com/vinsol-spree-contrib/spree-on-react).

1. Rails 3.2.7
2. Spree (3.1)
3. Spree AMS
4. CORS enabled

Run this app on port specified in .env file in spree-on-react project. This project expects spree-on-react server to be running on http://loalhost:3000. You may tweak this behaviour in configurations.
